# ADR-0003 — One Active and Many Archived Initiatives

- **Status:** Accepted
- **Date:** 2026-07-13

## Decision

A repository has at most one active initiative and any number of immutable closed or abandoned
archives. Continued work uses a successor with a new identity and provenance links; approval is
never inherited.

## Consequences

Current state stays unambiguous. Archived work cannot reopen or be mutated through supported
commands.
